# Expense Tracker

A simple, browser-based expense tracker to manage your income and expenses.

## Features

- Add income and expense transactions with a description, amount, and date
- View a live summary of total income, total expenses, and current balance
- Doughnut chart visualization of income vs expenses (powered by Chart.js)
- Transaction history with delete support
- Data persisted in browser `localStorage` — no backend required

## How to Use

1. Clone or download the repo
2. Open `index.html` in any modern browser
3. Add transactions using the form — select **Income** or **Expense**, enter a description, amount, and date, then click **Add Transaction**
4. Click the **✕** button on any transaction to delete it

## Tech Stack

- HTML, CSS, JavaScript (vanilla)
- [Chart.js](https://www.chartjs.org/) via CDN

## Live Preview

Open `index.html` directly in your browser — no server or build step needed.
